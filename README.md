# GeoChat
An HTML Discord-like webchat

To run, open the file in your web browser.
Once open, enter your prefered username in the box that says "Your Username" under "Identity".
Remember to press "Set".

Then, enter the username of the person you want to connect to in the box labled "Enter Peer ID", and press the button labled "Connect".
You can do this as many times as needed.
